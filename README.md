# python-cli-bazel-seg-sieve

## Description
Given a number find all primes.

## Tech stack
- python 3.8

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
